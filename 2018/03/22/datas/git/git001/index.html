  • Fast-Forward

    • (master) git merge hotfix
    • master는 hotfix 를 가리키게 된다. (별도의 커밋을 생성하지 않는다.)

  • Recursive Stragety

    • master 와 iss53 의 공통의 조상을 찾는다.
